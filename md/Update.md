## 插件更新日志

### 2023/1/24
- 2.1.1   
  1. 重做BanGUI判断逻辑    
    > [!note]
  > Player  玩家名  
  > time    时间  
  > Reason  原因    
  > #&#     为引号"  用于防止带空格的玩家封禁失败   
  > ```json
  > {
  >    "Ban_Cmd": {
  >         "Ban": "ban #&#${Player}#&# ${time} ${Reason}",
  >         "UnBan": "unban #&#${Player}#&#"
  >    }
  > }
  > ```

```json
  op: [//复制下面的xuid到新文件即可

    "6666xxxxxxxxxx"
  ]
}
```
